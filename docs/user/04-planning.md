# Planifier ses repas

## Acceder au planning

Cliquez sur **Planning** dans la barre laterale. Le planning s affiche sous forme de tableau avec les 7 jours de la semaine en colonnes et les 4 creneaux (Matin, Midi, Soir, Collation) en lignes.

## Ajouter une recette au planning

1. Cliquez sur la case **+ Ajouter** du creneau souhaite.
2. Selectionnez la recette dans le menu deroulant.
3. La recette apparait dans la case ; vous pouvez la retirer a tout moment.

## Navigation

- Utilisez les fleches gauche / droite pour changer de semaine.
- La semaine courante est mise en evidence.

## Generer la liste de courses

1. Remplissez votre planning de la semaine.
2. Cliquez sur **Generer la liste de courses** en haut a droite.
3. SUPMEAL agrege automatiquement tous les ingredients des recettes planifiees, ajuste les quantites selon les portions et cree une liste de courses.

Vous retrouverez vos listes de courses dans la section dediee (accessible depuis les parametres dans cette version, integration complete dans une prochaine mise a jour).
