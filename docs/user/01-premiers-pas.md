# Premiers pas

## Creer un compte

1. Rendez-vous sur http://localhost:5173 (URL par defaut en developpement).
2. Cliquez sur **Creer un compte**.
3. Renseignez votre nom (optionnel), nom d utilisateur, email et mot de passe (8 caracteres minimum).
4. Cliquez sur **Creer mon compte**. Vous etes connecte automatiquement.

## Se connecter

1. Sur la page de connexion, saisissez votre email et mot de passe.
2. Vous pouvez aussi utiliser un provider OAuth (Google, GitHub, Microsoft) si l administrateur les a configures.

## Interface principale

L interface se compose de :

- **Barre laterale gauche** : navigation principale (Tableau de bord, Mes cookbooks, Planning, Parametres).
- **En-tete** : message d accueil personnalise et bouton "Nouvelle recette".
- **Zone centrale** : contenu de la page active.

## Tableau de bord

Le tableau de bord affiche :

- Une **barre de recherche** pour trouver une recette par titre, ingredient ou tag.
- Un **filtre Favoris** pour ne voir que vos recettes preferees.
- Une **grille de tags** pour filtrer rapidement.
- La liste de vos **recettes recentes**.
- Vos **cookbooks partages** (avec le nombre de membres et de recettes).
