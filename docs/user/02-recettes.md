# Gerer ses recettes

## Creer une recette

1. Cliquez sur **Nouvelle recette** (en haut a droite).
2. Remplissez les champs :
   - **Titre** (obligatoire)
   - **Description** (optionnel)
   - **Portions**, temps de preparation, temps de cuisson
   - **Difficulte** (facile, moyen, difficile)
   - **Type de cuisine** (francaise, italienne, etc.)
   - **URL source** (si la recette vient d un autre site)
3. Ajoutez les **ingredients** : nom, quantite, unite. Utilisez le bouton + pour en ajouter d autres.
4. Ajoutez les **etapes** dans l ordre. Chaque etape peut etre sur plusieurs lignes.
5. Cliquez sur **Creer la recette**.

## Consulter / modifier une recette

- Cliquez sur une recette (depuis le tableau de bord ou un cookbook) pour voir son detail.
- Sur la page de detail, vous voyez ingredients, etapes, et eventuellement les commentaires (recettes de cookbook).
- Boutons disponibles : **Ajouter aux favoris**, **Planifier**, **Editer**, **Supprimer** (owner uniquement).

## Rechercher et filtrer

Depuis le tableau de bord :

- **Recherche** : tapez un mot-cle (titre, ingredient, tag). La recherche est instantanee.
- **Favoris** : cochez le filtre pour ne voir que vos favoris.
- **Tags** : cliquez sur un tag pour filtrer. Cliquez a nouveau pour desactiver.

Depuis un cookbook :

- **Recherche dans le cookbook** : tapez un mot-cle. Le resultat est limite aux recettes du cookbook.
- **Filtres** : tags, ingredients, temps max, favoris (dans l API, ces filtres sont aussi disponibles).

## Image de recette

Sur la page d edition d une recette, vous pouvez televerser une image (JPEG, PNG, WebP, GIF, max 5 Mo). L image s affiche en haut de la fiche recette.
